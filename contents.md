## 문제

기존의 텍스트 나열식 해몽 서비스들은 사용자가 꿈 분석을 요청하고 결과를 확인하는 과정이 단순히 지루하고 일회성 텍스트 확인에 그쳤습니다. 또한, 단일 인공지능 API 공급자(예: OpenAI)만을 고집하여 사용자의 API 키 관리 편의성이나 다양한 모델(Gemini, Claude 등)의 장점을 취사선택하기 어려웠습니다. 사용자가 자신의 꿈이라는 개인적이고 신비로운 무의식을 시각적·감성적으로 몰입감 있게 해석하고 보관하고 싶어 하는 요구가 있습니다.

---

## 사용자

자신이 꾼 밤 사이의 꿈에 담긴 무의식적이고 심리적인 깊은 의미를 찾고자 하는 일반인, 그리고 딱딱한 텍스트 기반 서비스 대신 2D 플랫폼 게임 및 레트로 RPG 스타일 대화 조작 방식에 흥미를 느끼는 사용자.

---

## 핵심 기능 (3개 이하)

1. **2D 플랫폼어 및 대화형 조작 인터페이스**: 플레이어가 마법사 캐릭터를 조작하여 숲속의 거대 수정구슬(예언자)에 다가가 상호작용하고, 대화의 진행(예/아니오 선택 및 감정 고르기) 또한 마우스 없이 키보드 방향키(`WASD`)와 `Enter`/`E`/`Ctrl+Enter` 단축키로 제어합니다.
2. **다중 AI API 통합 라우팅 (Multi-Provider Route)**: OpenAI(GPT-4o-mini), Google Gemini(Gemini 3.1 Flash Lite), Anthropic(Claude 3.5 Haiku) 및 커스텀 호환 API(DeepSeek, Grok 등)를 라우팅하여 구조화된 JSON 데이터 포맷으로 정밀한 꿈 요약과 해몽 리포트를 얻습니다.
3. **꿈 기억 보관소 (Dream Vault)**: 해몽 결과를 로컬 안전 영역(localStorage)에 시계열 카드 데이터로 자동 보관하며, 검색 필터 바를 활용해 과거의 꿈 기록과 해몽 처방전(행운의 색, 숫자, 행동 지침)을 손쉽게 찾아봅니다.

---

## 입력 → AI 처리 → 출력

사용자 꿈 서술 텍스트(최소 10자) 및 분위기 감정 입력 → 선택된 AI 엔진(OpenAI/Gemini/Claude 등)의 JSON 모드 호출 및 해몽가 프롬프트 기반 처리 → 신비로운 예언서 양식(꿈 요약, 심리 분석, 길흉 해석, 상징 키워드, 행운 아이템 처방)으로 화면에 출력 및 보관소 저장

---

## 기술 스택

- **UI**: Vanilla HTML5 Canvas, JS Module (ES6), Vanilla CSS (Cosmic Dark & Glassmorphism 디자인 적용)
- **모델 / API**: OpenAI API (GPT-4o-mini), Google Gemini API (gemini-3.1-flash-lite), Anthropic API (claude-3-5-haiku), Custom OpenAI-compatible APIs (DeepSeek, Grok, Ollama 등)
- **외부 서비스**: 브라우저 LocalStorage (API 설정 정보 암전 관리 및 꿈 해몽 이력 로컬 저장)
