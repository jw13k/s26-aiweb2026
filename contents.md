## 문제

기존의 텍스트 나열식 해몽 서비스들은 단순히 분석 결과를 확인하는 것에 그쳐, 무의식을 다루는 '꿈'이라는 주제에 걸맞은 몰입감을 주지 못했습니다. 또한 단일 AI에 의존하여 사용자 입맛에 맞는 모델을 선택하거나 개인 API 키를 활용하기 어려웠습니다. 이에 3D 공간과 흑백 기반의 레트로 OS 환경이라는 독특한 인터페이스를 통해, 사용자가 자신의 내면을 안전하고 은밀하게 탐구한다는 감각을 극대화하고자 합니다.

---

## 사용자

단순한 텍스트 기반 서비스에 지루함을 느끼고, 방을 돌아다니며 컴퓨터를 켜는 몰입감 있는 인터페이스(방탈출 혹은 쯔꾸르 감성)에 흥미를 느끼는 사람. 자신이 꾼 꿈의 심리적·상징적 의미를 감성적인 시구절과 함께 보관하고 싶은 모든 사용자.

---

## 핵심 기능 (3개 이하)

1. **1인칭 3D 공간과 가상 OS 인터페이스**: Vanilla JS와 Canvas만으로 구현된 3D 방을 키보드(WASD)와 마우스로 탐험합니다. 데스크탑에 다가가 상호작용하면 미니멀리즘 흑백 디자인의 'DREAMEan OS'가 열리며, 다중 창(Window) 시스템을 기반으로 앱들을 실행합니다.
2. **다중 AI 해몽 및 감성적 출력**: 설정 앱에서 Gemini, OpenAI, Claude 등 원하는 AI 엔진을 자유롭게 전환할 수 있습니다. 프록시 서버를 통해 API 키가 안전하게 보호되며, 무서운 악몽조차 치유와 성장의 과정으로 따뜻하게 승화시킨 심리 분석, 상징 해석, 행운의 처방전, 시구절(Quote)이 포함된 구조화된 리포트를 제공받습니다.
3. **꿈 보관소와 이스터에그**: 사용자의 해몽 기록은 로컬 브라우저에 시계열로 자동 보관되며, 언제든 다시 열람하고 재해석할 수 있습니다. 또한, 바탕화면 스마일 눈동자에 포인터 잠금(Pointer Lock) API를 활용한 '핑퐁(Pong)' 미니게임 이스터에그를 숨겨두어 재미 요소를 더했습니다.

---

## 입력 → AI 처리 → 출력

사용자 이동 및 상호작용(WASD, E 키) → 가상 OS 내 '꿈 기록' 앱에서 한국어 꿈 내용 작성 → 선택된 AI 엔진으로 프롬프트(감정 톤, 구조화 JSON 포맷 지시) 전송 → 응답 데이터를 예언서 양식(요약, 심리학/신비학 분석, 상징물, 행운 요소, 시구절)의 HTML 리포트로 화면에 렌더링 및 로컬 보관소에 영구 저장.

---

## 기술 스택

- **UI / Frontend**: Vanilla HTML5, Canvas 2D API (Pseudo-3D Rendering), Custom CSS (미니멀리즘 흑백 OS 디자인, Custom SVG 커서)
- **Backend / Security**: Node.js (Express 프록시 서버, Rate Limiting, CORS 및 Referer 검증, 페이로드 제한을 통한 API 키 보안)
- **AI / LLM API**: Google Gemini (3.1-flash-lite), OpenAI (gpt-4o-mini), Anthropic (claude-3-haiku) 등
- **데이터베이스**: 브라우저 LocalStorage (API 키 및 꿈 해몽 데이터 저장)
